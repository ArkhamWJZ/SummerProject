## 后端接口文档

1. /imgProcess/upload

接口描述：利用post请求该接口，上传用户验证码图片，返回验证码结果

请求参数：[

    "img":用户上传的图片
    
]

返回参数：[
    
    "code":200,结果成功；400,结果失败
    
    "imgCode":验证码识别结果，str类型

]

2. /imgProcess/createImg

接口描述：请求该接口，获取随机一张验证码照片和验证码结果

请求参数：无

返回参数：[

    "code":200
    
    "img":图片路径，为相对路径
    
    "imgCode":验证码结果
    
]
